# PLAN3 — Следующие шаги

## 1. Расширение базы знаний

**Проблема:** RAGAS показал низкий faithfulness на темах механизмов прерывания контакта.

| Вопрос | faithfulness | context_precision |
|---|---|---|
| Интроекция | 0.532 | 0.000 |
| Незавершённый гештальт | 0.531 | 0.750 |

Поиск возвращает чанки про стыд/социализацию вместо интроекции — тема не покрыта в двух имеющихся книгах.

### Книги для добавления

| Книга | Что закрывает | Приоритет |
|---|---|---|
| **Польстер «Интегрированная гештальт-терапия»** (Polster — Gestalt Therapy Integrated) | Интроекция, проекция, ретрофлексия, дефлексия, слияние — каждый механизм отдельной главой | 🔴 высокий |
| **Перлз «Гештальт-терапия»** (Perls, Hefferline, Goodman, 1951) | Первоисточник: незавершённый гештальт, фигура/фон, осознавание | 🔴 высокий |
| **Кларксон «Гештальт-консультирование в действии»** (Clarkson — Gestalt Counselling in Action) | Практический разбор каждого механизма с клиническими примерами | 🟡 средний |

### Шаги
1. Найти PDF книг (английские издания)
2. `POST /api/admin/files/upload` — загрузить
3. `POST /api/admin/files/convert?mode=smart` — конвертировать
4. `POST /api/admin/files/ingest?mode=smart` — ingest в Qdrant
5. Проверить: `GET /api/admin/search?query=introjection&top_k=5`
6. Запустить RAGAS и сравнить с текущим (faithfulness: 0.865, context_precision: 0.786)

## 3. Голосовые сообщения

Терапевт может надиктовать вопрос прямо с сессии.

**Как работает:**
1. Пользователь отправляет голосовое в Telegram
2. Gemini Audio API транскрибирует аудио → текст
3. Дальше стандартный RAG пайплайн

**Что нужно:**
- Добавить обработчик `MessageHandler(filters.VOICE)` в `handlers.py`
- Скачать `.ogg` через `bot.get_file()`, передать в Gemini Files API
- Gemini возвращает транскрипцию → передаём в `rag_ask()`
